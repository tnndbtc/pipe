### 9b — RenderPlan.json

Build these lookup dicts from AssetManifest.{locale}.json:
```python
# Strip "pack-" prefix from pack_id to get character_id
char_map  = { c["pack_id"].removeprefix("pack-"): c["asset_id"]
              for c in manifest["character_packs"] }
bg_map    = { bg["bg_id"]: bg["asset_id"]
              for bg in manifest["backgrounds"] }
vo_map    = { v["item_id"]: v
              for v in manifest["vo_items"] }
sfx_map   = { s["item_id"]: s
              for s in manifest.get("sfx_items", []) }
music_map = { m["item_id"]: m
              for m in manifest.get("music_items", []) }

# Resolved-asset lookup: asset_id/item_id → is_placeholder
media_map = { item["asset_id"]: item
              for item in manifest["resolved_assets"] }
```

For each shot in ShotList.shots[] (preserve order):
  duration_ms = int(shot.duration_sec × 1000)

  vo_lines: for each id in shot.audio_intent.vo_item_ids
    where vo_map[id] exists AND media_map.get(id, {}).get("is_placeholder", True) == False:
    Distribute timeline offsets evenly across all assigned lines:
      slot = duration_ms / len(assigned_lines)
      line N: timeline_in_ms  = int(N × slot)
              timeline_out_ms = int((N+1) × slot)
    Each entry:
    ```json
    {
      "line_id"         : "<v.item_id>",
      "speaker_id"      : "<v.speaker_id>",
      "text"            : "<v.text>",
      "timeline_in_ms"  : <int>,
      "timeline_out_ms" : <int>
    }
    ```
    Verify: every timeline_out_ms ≤ duration_ms.
    Use [] if no ids or all have is_placeholder == true.

  sfx_asset_ids: for each id in shot.audio_intent.sfx_item_ids ([] if field absent)
    include media_map[id]["asset_id"] only where sfx_map[id] exists
    AND media_map.get(id, {}).get("is_placeholder", True) == False
    Result is [] if no ids or all have is_placeholder == true.

  music_asset_id:
    mid = shot.audio_intent.music_item_id  (null if field absent)
    if mid exists AND media_map.get(mid, {}).get("is_placeholder", True) == False:
      use media_map[mid]["asset_id"]
    else: null
