# Task: AI Narrative Pipeline — Story to AssetManifest.json

You are the writing agent, world engine, and canon guardian for a narrative
pipeline.  Your job is to take a story title/direction and produce all JSON
artifacts from StoryPrompt.json through AssetManifest.json, then update
canon.json.  Write no source code.  Write JSON files directly to disk.

---

## Progress tracking — mandatory stdout output

**First action of all:** before reading any file, print the pipeline plan:

```
══════════════════════════════════════════════════════════════
  PIPELINE PLAN  —  10 stages  (step 9 is conditional)
══════════════════════════════════════════════════════════════
  [0]  Read story file & populate variables
  [1]  Canon check
  [2]  Produce StoryPrompt.json
  [3]  Produce Script.json
  [4]  Produce ShotList.json
  [5]  Produce AssetManifest_draft.shared.json + AssetManifest_draft.en.json
  [6]  Write canon_diff.json
  [7]  Update canon.json
  [8]  Produce locale variants  (one pass per non-en locale)
  [9]  AssetManifest_final.json + RenderPlan.json  ← conditional
══════════════════════════════════════════════════════════════
```

Then, throughout execution:
  • Print `▶ [N/9] <step title>  …` the moment you begin each step.
  • Print `✓ [N/9] <step title>  done` immediately after all files for that
    step have been written and schema-validated.
  • Step 9: if skipped, print the skip message defined in that section instead
    of the done line.

## Step 4 — Produce ShotList.json

**Print:** `▶ [4/9] Step 4 — Produce ShotList.json  …`

Write to: <episode_dir>/ShotList.json

Required fields (ShotList.v1.json — inspect the schema first):
  schema_id, schema_version, shotlist_id, script_ref, timing_lock_hash, shots[]

Each shot must have:
  shot_id, scene_id, duration_sec, camera_framing, camera_movement,
  characters[], background_id, audio_intent

background_id: set to the bg_id of the background for this shot's location.
  You know all scene locations while writing the ShotList, so assign
  background_ids now — before the manifest exists. The bg_id is the slug
  you will use as the "bg_id" key in AssetManifest_draft.backgrounds[] in
  Step 5. Format: "bg-<location-slug>"  (e.g. "bg-karnak-inner-sanctuary").
  Multiple scenes may share the same background_id (e.g. archives used in
  both s02 and s05).

audio_intent must include:
  sfx_tags[]    : list of sound-effect description strings for this shot
  music_mood    : one-phrase music direction, or null if silent
  vo_speaker_id : character_id of the speaking character, or null
  vo_text       : verbatim dialogue line, or null
  vo_item_ids   : ordered list of item_id values this shot's VO lines will
                  carry in AssetManifest vo_items[] (Step 5).
                  Format: ["vo-<scene_id>-<NNN>", ...]
                  Use an empty array [] if no VO in this shot.
                  Assign every vo_item in the episode to exactly one shot —
                  no VO line should be orphaned. Distribute multi-line
                  conversations across the shots in which they are spoken.
                  (This array creates the explicit shot→VO links the plan
                  builder uses in Step 9b to populate shots[].vo_lines[].)
  sfx_item_ids  : ordered list of item_id values for this shot's SFX,
                  parallel with sfx_tags[] (one id per tag, same order).
                  Format: ["sfx-<shot_id>-<NNN>", ...]  (NNN zero-padded,
                  e.g. "sfx-s01-sh01-001").
                  Use an empty array [] if the shot has no SFX.
                  (These IDs are the explicit shot→SFX links consumed by
                  Step 5 and Step 9b.)
  music_item_id : "music-<shot_id>" when music_mood is not null, else null.
                  (This ID is the explicit shot→music link consumed by
                  Step 5 and Step 9b.)

timing_lock_hash: sha256 of the sorted shot_id list (use Python to compute).
Total duration should be 60–120 seconds.

Validate against ShotList.v1.json before proceeding.

**Print:** `✓ [4/9] Step 4 — Produce ShotList.json  done`

## Step 5 — Produce AssetManifest drafts (shared + locale)

**Print:** `▶ [5/9] Step 5 — Produce AssetManifest drafts  …`

Produce TWO files:

### File 1 — Shared manifest (locale-free assets)

Write to: <episode_dir>/AssetManifest_draft.shared.json

Required top-level fields:
  schema_id       : "AssetManifest"
  schema_version  : "1.0.0"
  manifest_id     : "<project_id>-<episode_id>-shared-manifest"
  project_id, episode_id, shotlist_ref : "ShotList.json"
  locale_scope    : "shared"
  vo_items        : []   ← always empty

Derive from ShotList.json:
  - character_packs : one entry per unique character (license_type: "proprietary_cleared")
  - backgrounds     : one entry per unique background_id (license_type: "proprietary_cleared")

For every character_pack and background entry, also populate:
  - search_prompt  : 5-10 keywords for free library image search
  - search_filters : metadata criteria (orientation, media_type, min_width/height,
                     color_mood, category — same rules as before)
  - ai_prompt      : detailed text-to-image prompt
  - motion         : null for static; { type, description, participants, duration_sec, style } for animated

sfx_items[] — one entry per (shot, sfx_tag) pair:
  item_id, shot_id, tag (expanded), duration_sec, license_type: "proprietary_cleared"
  Skip shots where sfx_item_ids is absent or [].

music_items[] — one entry per shot with non-null music_item_id:
  item_id, shot_id, music_mood, duration_sec, license_type: "proprietary_cleared"
  Also author these optional timing fields (editorial intent):
    start_sec : when music starts within the shot (almost always 0.0)
    duck_db   : attenuation in dB during VO (e.g. -12.0; use -18.0 for intimate scenes;
                0.0 for action/ambience-only shots with no VO)
    fade_sec  : ramp duration at duck boundaries (e.g. 0.15)
  Note: duck_intervals are NOT included here — computed later by manifest_merge.py.
  Skip shots where music_item_id is null or absent.

### File 2 — Locale manifest (English VO only)

Write to: <episode_dir>/AssetManifest_draft.en.json

Required top-level fields:
  schema_id        : "AssetManifest"
  schema_version   : "1.0.0"
  manifest_id      : "<project_id>-<episode_id>-en-manifest"
  project_id, episode_id, shotlist_ref : "ShotList.json"
  locale           : "en"
  locale_scope     : "locale"
  shared_ref       : "AssetManifest_draft.shared.json"
  character_packs  : []
  backgrounds      : []
  background_overrides : []   ← always empty at draft time
  vo_items         : [...]    ← English VO items

vo_items[] — one entry per dialogue line:
  item_id, speaker_id, text (English), license_type: "proprietary_cleared"
  tts_prompt: copy ALL azure_* fields verbatim from VoiceCast.json character["en"]:
    { locale: "en", azure_voice, azure_style (omit if null), azure_style_degree,
      azure_rate, azure_pitch, azure_break_ms }
  VoiceCast.json is the sole TTS authority. Do NOT invent voice_style/emotion/pace.
  DO NOT add start_sec, end_sec, or duration_sec — written by post_tts_analysis.py.

Validate both artifacts against AssetManifest.v1.json before proceeding.

**Print:** `✓ [5/9] Step 5 — Produce AssetManifest drafts  done`

## Step 9 — Produce AssetManifest_final.json and RenderPlan.json

**Print:** `▶ [9/9] Step 9 — AssetManifest_final + RenderPlan  …`

Prerequisite: <episode_dir>/AssetManifest.media.json must exist.
If it does not exist, stop here and print:
  "✗ [9/9] Step 9 skipped — waiting for media-agent.
   Place AssetManifest.media.json in <episode_dir> and re-run from Step 9."

### 9a — AssetManifest_final.json

Write to: <episode_dir>/AssetManifest_final.json

Merge the draft envelope with the resolved items from the media manifest:
  schema_id      : "AssetManifest_final"
  schema_version : "1.0.0"
  manifest_id    : copied from AssetManifest_draft.shared.json → manifest_id
  project_id     : copied from AssetManifest_draft.shared.json → project_id
  shotlist_ref   : copied from AssetManifest_draft.shared.json → shotlist_ref
  items[]        : passthrough from AssetManifest.media.json → items[]
                   (includes all asset types: character, background, vo,
                   sfx, music). Pass all items through unchanged.
                   Skip any item where is_placeholder == true —
                   include only items where is_placeholder == false.

Validate against AssetManifest_final.v1.json before proceeding.

### 9b — RenderPlan.json

Write to: <episode_dir>/RenderPlan.json

  schema_id         : "RenderPlan"
  schema_version    : "1.0.0"
  plan_id           : "plan-{{PROJECT_SLUG}}-{{EPISODE_ID}}"
  project_id        : "{{PROJECT_SLUG}}"
  manifest_ref      : AssetManifest_final.manifest_id
  timing_lock_hash  : copied from ShotList.json → timing_lock_hash
  profile           : "preview_local"
                      (override to "draft_720p" or "final_1080p" if story.txt
                       contains a "Render profile" field)
  resolution        : "1280x720"
  aspect_ratio      : "16:9"
  fps               : 24
  resolved_assets[] : flatten AssetManifest_final.items[], one entry per item:
    {
      "asset_id"      : item.asset_id,
      "asset_type"    : item.asset_type,
      "uri"           : item.uri,
      "license_type"  : item.metadata.license_type,
      "is_placeholder": item.is_placeholder
    }
  shots[]           : ordered shot sequence for the renderer. Provides explicit
                      per-shot asset assignments and VO lines; replaces the
                      background-grouping heuristic previously used by the renderer.

                      Build lookup dicts from AssetManifest_draft.shared.json
                      and AssetManifest.media.json:
                        char_map  = { c.character_id : c.asset_id }
                                    (derive character_id by stripping "pack-" prefix
                                     from each character_pack.pack_id)
                        bg_map    = { bg.bg_id : bg.asset_id }
                                    for each entry in backgrounds[]
                        vo_map    = { v.item_id : v }
                                    for each entry in vo_items[]
                        sfx_map   = { s.item_id : s }
                                    for each entry in sfx_items[] (empty if absent)
                        music_map = { m.item_id : m }
                                    for each entry in music_items[] (empty if absent)
                        media_map = { item.asset_id : item }
                                    for each entry in AssetManifest.media.json items[]

                      For each shot in ShotList.shots[] (preserve array order):
                        shot_id             : shot.shot_id
                        scene_id            : shot.scene_id
                        duration_ms         : int(shot.duration_sec × 1000)
                        background_asset_id : bg_map[shot.background_id]
                                             (null if background_id is null or missing)
                        character_asset_ids : [ char_map[c.character_id]
                                               for c in shot.characters ]
                                             (skip characters not found in char_map)
                        vo_lines            : for each id in shot.audio_intent.vo_item_ids
                                               (or [vo_item_id] if only deprecated field present)
                                               where vo_map[id] exists
                                               AND media_map.get(id).is_placeholder == false:
                                               Distribute timeline offsets evenly:
                                                 slot = duration_ms / len(assigned_lines)
                                                 line N: timeline_in_ms  = N * slot
                                                         timeline_out_ms = (N+1) * slot
                                               Each entry:
                                               {
                                                 "line_id"        : v.item_id,
                                                 "speaker_id"     : v.speaker_id,
                                                 "text"           : v.text,
                                                 "timeline_in_ms" : (computed above),
                                                 "timeline_out_ms": (computed above)
                                               }
                                             Empty array [] if no ids or all
                                             have is_placeholder == true.
                                             Plan builder must verify every timeline_out_ms
                                             ≤ duration_ms before writing.
                                             Phase 0: even distribution approximation.
                                             Phase 1: replace with real TTS timestamps.
                        sfx_asset_ids       : for each id in shot.audio_intent.sfx_item_ids
                                               ([] if field absent)
                                               where sfx_map[id] exists
                                               AND media_map.get(id).is_placeholder == false:
                                               include media_map[id].asset_id
                                             Result is [] if no ids or all
                                             have is_placeholder == true.
                        music_asset_id      : mid = shot.audio_intent.music_item_id
                                             if mid exists AND
                                             media_map.get(mid).is_placeholder == false:
                                               use media_map[mid].asset_id
                                             else: null

Validate against RenderPlan.v1.json before writing.

**Print:** `✓ [9/9] Step 9 — AssetManifest_final + RenderPlan  done`
