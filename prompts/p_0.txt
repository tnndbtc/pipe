Your ONLY job: read a story file, extract variables, write two output
files, create directories, and print a confirmation block.
Do NOT generate any story content.

## Task

Print: `▶ [0/9] Stage 0 — Read story file  …`

Read the story file at path: {{STORY_FILE}}

Extract these variables (the file may use any human-readable format —
use judgement to map content to each field):

| Variable          | Source rule                                                                |
|-------------------|----------------------------------------------------------------------------|
| STORY_TITLE       | "Story title", "Title", or first prominent title line                      |
| EPISODE_NUMBER    | "Episode", "Episode number", or integer near "episode"                     |
| EPISODE_ID        | "Episode id" verbatim (e.g. "s01e02"); fall back to ep<zero-padded num>   |
| EPISODE_DIRECTION | "Direction", "Synopsis", "Summary", or full narrative description block    |
| LOCALES           | "Locales" / "Languages" field; default to "en" if absent                  |
| PROJECT_SLUG      | "Project slug" if present; else slugify STORY_TITLE (lowercase, hyphens)  |
| SERIES_GENRE      | "Genre"; infer from tone/setting if not explicit                           |
| GENERATION_SEED   | "Seed" if present; else compute int(sha256(STORY_TITLE+EPISODE_ID)[:8])   |
| RENDER_PROFILE    | "Render profile"; default "preview_local" if absent                        |

After extraction, print this exact confirmation block to stdout:

